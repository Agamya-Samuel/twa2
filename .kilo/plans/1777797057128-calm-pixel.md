# Plan: Match Admin Page Width to Header Width

## Problem

## Solution
Modify the Admin page container to use the same width and padding classes as the Header component.

## Changes Needed
1. Update the container div in `src/app/admin/page.tsx` to match the Header's container classes:

   - Change to: `<div className="mx-auto max-w-7xl px-4 sm:px-6 lg:px-8 py-8">`

## Implementation Details
